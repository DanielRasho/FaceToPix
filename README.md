# FaceToPix

All scripts under this repo are snippets of code to locally test before
uploading them to Collab and doing them for real.

[Check out the final report](https://colab.research.google.com/drive/1TD5hKATc56rh4ZRiudcNZirpvtZPyvPo?usp=sharing)

In the report we mention all the architectures we tried, including the original
code from the CycleGAN paper. Checkout the notebooks directory to see the code
for all this attempts.

- **Proyecto FaceToPix**: Notebook with our CycleGAN implementation.
- **pix2pix**: Notebook with the Paper CycleGAN implementation.
- **StyleTransfer**: StyleTransfer architecture for generating pixel art from
  images.
